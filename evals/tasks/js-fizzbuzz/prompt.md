Create fizzbuzz.js as an ES module that exports a function `fizzbuzz(n)`.

It returns an array for the numbers 1 through n where:
- multiples of 3 become the string "Fizz"
- multiples of 5 become the string "Buzz"
- multiples of both 3 and 5 become "FizzBuzz"
- every other number stays as the number itself

Example: fizzbuzz(5) returns [1, 2, "Fizz", 4, "Buzz"].
